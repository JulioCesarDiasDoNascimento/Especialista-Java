# Anotações

Principal ponto code completion!!
---

> Live Templates (precisa colocar as chaves)
> * main - cria o metodo main
> * sout - cria o System.out.println()
> * soutv - cria o System.out.println() de uma variavel
> * fori - cria um for com incremento
> * iter - cria um for each
> * if - cria um if
> * trycatch - cria um bloco try catch
> * switch - cria um switch n
> * while - cria um while
> * psvm - cria o método main (sem as chaves)
---

* Postfix Completion, bem interessante!
